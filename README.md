# 🌐 VLAN-Subnetting-Mentor: Interactive Training Platform

פרויקט גמר זה מציג **אפליקציה ומשחק חינוכי אינטראקטיבי** מבוסס בינה מלאכותית, שמטרתו לתרגל ולבחון הבנה לוגית של רשתות תקשורת בתחומי ה-VLAN וה-Subnetting.

הפרויקט פותח בגישת **Vibe Coding** – פיתוח מונחה AI המעביר את הפוקוס מכתיבת קוד סינטקטי מסובך, לתכנון ארכיטקטורה, חוויית משתמש חכמה ותוכן פדגוגי איכותי.

---

## 🎯 מבנה המערכת
הפרויקט משלב בין שני אלמנטים מרכזיים המנוהלים ב-GitHub:

* **האפליקציה והמשחק החינוכי:** סימולציית תרחישים אינטראקטיבית מבוססת ענן המציבה את הסטודנט מול מקרי מבחן ותיאורי רשת אמיתיים.
* **סוכן ה-GEM – "יועץ: VLAN & Subnetting":** ה"מנוע" של האפליקציה. סוכן AI המנהל את שלבי הסימולציה, בוחן את תשובות הסטודנט ומספק הנחיה תאורטית לעידוד חשיבה עצמאית.

## 🛠 ניהול וארגון הפרויקט (GitHub)
מאגר זה (`VLAN-Subnetting-Mentor`) משמש כבסיס הניהולי והארכיטקטוני:
* **תיעוד הלוגיקה:** הגדרת חוקי המשחק ותרחישי הרשת בשפה טבעית.
* **ניהול גרסאות:** מעקב אחר חומרי הרקע ובסיס הידע הטכני עליו מאומן הסוכן.

## 🚀 כיצד פועל המשחק?
1. **מתחילים תרחיש:** המערכת מציגה משימת רשת (למשל: תכנון רשת VLAN מופרדת).
2. **אתגר וחשיבה:** הסטודנט מציע פתרון או תוכנית פעולה.
3. **משוב והנחיה:** הסוכן מנתח את התשובה ומקדם את הסטודנט לשלב הבא, או מכוון אותו לתיקון טעויות לוגיות.
