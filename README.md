# 🌐 Network Master: Interactive VLAN & Subnetting Trainer

פרויקט גמר זה מציג **משחק ואפליקציה חינוכית אינטראקטיבית** המבוססת על בינה מלאכותית, שמטרתה לתרגל ולבחון הבנה לוגית של רשתות תקשורת (VLANs & Subnetting). 

הפרויקט פותח בגישת **Vibe Coding** – פיתוח מונחה AI המעביר את הפוקוס מכתיבת קוד סינטקטי מסובך, לתכנון ארכיטקטורה וחוויית משתמש חכמה.

## 🎯 מבנה המערכת (שילוב האפליקציה והסוכן)
הפרויקט משלב בין שני אלמנטים מרכזיים המנוהלים ומסוכמים ב-GitHub:

1. **האפליקציה והמשחק החינוכי:** במקום לבנות ממשק קוד כבד (Python) שנוטה לבאגים, המשחק הומר לסימולציית תרחישים מבוססת ענן. המשחק מציב את הסטודנט מול מקרי מבחן אמיתיים של רשתות תקשורת (כמו הגדרת טבלאות ניתוב או חלוקת Subnets).
2. **סוכן ה-GEM (המנטור החכם):** סוכן AI מותאם אישית שמשמש כ"מנוע" של המשחק. הוא מנהל את שלבי המשחק, בוחן את תשובות הסטודנט, ומספק רמזים והסברים תאורטיים (ללא מתן פתרון ישיר) כדי להוביל לחשיבה עצמאית.

## 🛠 ניהול וארגון הפרויקט (GitHub)
מאגר זה (Repository) משמש כבסיס הניהולי של הפרויקט:
* **תיעוד הלוגיקה:** הגדרת חוקי המשחק ותרחישי הרשת בשפה טבעית.
* **ניהול גרסאות:** מעקב וארגון של חומרי הרקע ובסיס הידע הטכני עליו מאומן הסוכן.

## 🚀 כיצד פועל המשחק?
1. **מתחילים תרחיש:** המערכת (הסוכן) מציגה לסטודנט משימת רשת (למשל: תכנון רשת VLAN מופרדת לארגון).
2. **אתגר וחשיבה:** הסטודנט מציע פתרון או פקודות הגדרה.
3. **משוב מהמנטור:** ה-Gem מנתח את התשובה ומקדם את הסטודנט לשלב הבא במשחק, או מחזיר אותו לחשוב על טעויות לוגיות באמצעות שאלות מכוונות.
